## 프로그램 개요

**Stamily(스타밀리)**는 방과후 수강신청 및 후기 작성, 일정 관리 프로그램입니다.

Stamily란 라틴어로 '안정'의 뜻을 가진 'stabilitas'와 '함께, 연결'의 의미를 가진 'assimilate'를 합친 단어로,

일정관리를 통해 얻을 수 있는 안정감과 선후배, 선생님 간의 커뮤니티를 통해 얻을 수 있는 유대감을 표현하였습니다.
